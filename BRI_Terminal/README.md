# BRI Terminal

Static GitHub Pages build of the public BRI Archive Terminal.

Upload this folder's contents into a GitHub Pages repository folder, for example:

```text
archive-terminal/
  index.html
  terminal.css
  terminal.js
  archive-content.js
  data.js
  assets/
```

Then open:

```text
https://YOUR-USERNAME.github.io/YOUR-REPO/archive-terminal/
```

This build intentionally excludes the local Directorate/editor server files.
